# M.O.L.E 한국어 패치 v1.0.0

M.O.L.E (Steam) 전체 한국어 번역 패치입니다. **게임 정식 소유자 전용** · 비공식 팬 패치.

## 설치 방법 (개발 지식 필요 없음 · 약 2분)

**1단계 — 파일 받기**
아래 Assets에서 **`Mole-Windows_P.pak`** 를 클릭해 다운로드합니다. (압축파일 아님 — 받은 그대로 사용)

**2단계 — 게임 폴더 열기** (경로를 직접 찾을 필요 없습니다)
1. Steam 라이브러리에서 **M.O.L.E** 우클릭 → **관리** → **로컬 파일 보기**
2. 열린 창에서 폴더를 따라 들어갑니다: **`Windows`** → **`Mole`** → **`Content`** → **`Paks`**
3. 이 **`Paks`** 폴더 안에 받은 **`Mole-Windows_P.pak`** 파일을 끌어다 놓습니다.

**3단계 — ⚠️ 게임 언어를 "일본어"로 설정** (제일 중요!)
게임 실행 → **설정(Settings)** → **언어(Language)** → **`日本語`(일본어)** 선택
> 🔸 **"한국어"라는 항목은 목록에 없습니다.** 한국어를 일본어 자리에 넣은 방식이라, **`日本語`를 골라야 한글이 나옵니다.** ("日本語 = 한국어"라고 생각하세요.)

**완료!** 제거하려면 넣었던 `Mole-Windows_P.pak` 파일만 삭제하면 됩니다. (원본을 덮어쓰지 않아 안전합니다.)
