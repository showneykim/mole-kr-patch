# M.O.L.E 한국어 패치 v1.0.0

M.O.L.E (Steam) 전체 한국어 번역 패치입니다. **게임 정식 소유자 전용** · 비공식 팬 패치.

## 설치 (초간단)
1. 아래 `Mole-Windows_P.pak` 를 다운로드합니다.
2. 게임 폴더의 `Paks` 디렉터리에 넣습니다:
   `...\steamapps\common\M.O.L.E\Windows\Mole\Content\Paks\`
3. 게임을 실행하고 **설정 → 언어 → 日本語(일본어)** 를 선택합니다.
4. 끝! 제거하려면 `Mole-Windows_P.pak` 파일을 삭제하면 됩니다.

> 왜 일본어를 선택하나요? 이 패치는 게임이 안 쓰는 **일본어 로케일 슬롯에 한국어를 넣는** 방식이라, 게임에서 "일본어"를 고르면 한국어가 표시됩니다. 자세한 원리는 저장소 `docs/` 참고.
